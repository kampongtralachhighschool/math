សៀវភៅជំនួយស្មារតី៖ ប្រូបាប (ថ្នាក់ទី១២)

ចងក្រងតាមក្បួនស្តង់ដាប្រឡងបាក់ឌុប មានឧទាហរណ៍ លំហាត់អនុវត្តន៍ និងគន្លឹះដោះស្រាយ

ក្បួនទី ១៖ ការទាញយកព្រមគ្នា (ប្រើរូបមន្តបន្សំ)

📘 រូបមន្ត និង គោលការណ៍

នៅពេលលំហាត់មានពាក្យថា "ទាញយកព្រមគ្នា" ឬ "ក្នុងពេលតែមួយ" យើងត្រូវប្រើរូបមន្តបន្សំ៖

$C(n, k) = C_n^k = \frac{n!}{k!(n-k)!}$

ប្រូបាបនៃព្រឹត្តិការណ៍ $A$: $P(A) = \frac{\text{ចំនួនករណីស្រប}}{\text{ចំនួនករណីអាច}}$

🎯 ឧទាហរណ៍គំរូ

ក្នុងថង់មួយមានបាល់ក្រហម ៥ និងបាល់ខៀវ ៤។ គេចាប់យកបាល់ ៣ ព្រមគ្នា។ គណនាប្រូបាបដែលគេចាប់បានបាល់ក្រហម ២ និងខៀវ ១។

ដំណោះស្រាយ៖

  • ចំនួនបាល់សរុប: $5 + 4 = 9$ ។ ចាប់ ៣ ព្រមគ្នា។
  • ករណីអាច: $n(S) = C(9,3) = \frac{9!}{3!6!} = 84$
  • តាង $A$ ជាព្រឹត្តិការណ៍បាន ក្រហម២ និងខៀវ១។ ករណីស្រប: $n(A) = C(5,2) \times C(4,1) = 10 \times 4 = 40$
  • ប្រូបាប: $P(A) = \frac{40}{84} = \frac{10}{21}$

✍️ លំហាត់អនុវត្តន៍ (ដូរលេខ)

លំហាត់ ១.១៖ ថង់មួយមានបាល់ស ៦ និងខ្មៅ ៣។ ទាញយក ៣ ព្រមគ្នា។ រកប្រូបាបបានបាល់ស ២ និងខ្មៅ ១។

💡 Tip: $C(9,3)$ ជាភាគបែង, ភាគយកគឺ $C(6,2) \times C(3,1)$។
🔑 ចម្លើយ: $P = \frac{15 \times 3}{84} = \frac{45}{84} = \frac{15}{28}$

លំហាត់ ១.២៖ ប្រអប់មួយមានប៊ិចបៃតង ៤ និងលឿង ៥។ ចាប់យក ៣ ព្រមគ្នា។ រកប្រូបាបបានប៊ិចបៃតងទាំង ៣។

💡 Tip: រកតែបៃតង មិនបាច់គុណពណ៌លឿងទេ។
🔑 ចម្លើយ: $P = \frac{C(4,3)}{C(9,3)} = \frac{4}{84} = \frac{1}{21}$

លំហាត់ ១.៣៖ ក្រុមមួយមានបុរស ៧នាក់ និងស្ត្រី ៣នាក់។ គេរើសសមាជិក ៤នាក់ព្រមគ្នា។ រកប្រូបាបបានបុរស ២ និងស្ត្រី ២។

💡 Tip: ករណីអាច $C(10,4)$។
🔑 ចម្លើយ: $P = \frac{C(7,2) \times C(3,2)}{C(10,4)} = \frac{21 \times 3}{210} = \frac{63}{210} = \frac{3}{10}$

ក្បួនទី ២៖ ការទាញបន្តបន្ទាប់ មិនដាក់ចូលវិញ (ច្បាប់គុណ)

📘 រូបមន្ត និង គោលការណ៍

ពាក្យ "ជាបន្តបន្ទាប់ និងមិនដាក់ចូលវិញ" មានន័យថា ចំនួនសរុប (ភាគបែង) ត្រូវថយចុះម្ដង ១។ បើសំណួរប្រាប់ពីលំដាប់ (ឧ. បានពណ៌ ក រួច ខ) ប្រើច្បាប់គុណ។

P(A បន្ទាប់មក B) = $P(A) \times P(B|A)$

🎯 ឧទាហរណ៍គំរូ

ប្រអប់មួយមានបាល់ខ្មៅ ៤ និងក្រហម ៣។ គេទាញយកបាល់ ២ ជាបន្តបន្ទាប់មិនដាក់វិញ។ រកប្រូបាបដែលទាញបាន "បាល់ទី១ខ្មៅ និងទី២ក្រហម"។

ដំណោះស្រាយ៖

  • សរុបមាន ៧បាល់។
  • ទាញទី១ បានខ្មៅ: $P(B_1) = \frac{4}{7}$ (សល់បាល់ ៦ ក្នុងប្រអប់)
  • ទាញទី២ បានក្រហម: $P(R_2) = \frac{3}{6}$
  • ប្រូបាបសរុប: $P = \frac{4}{7} \times \frac{3}{6} = \frac{12}{42} = \frac{2}{7}$

✍️ លំហាត់អនុវត្តន៍ (ដូរលេខ)

លំហាត់ ២.១៖ ថង់មួយមានបាល់ខៀវ ៥ និងក្រហម ៤។ ទាញបាល់ ២ បន្តបន្ទាប់មិនដាក់វិញ។ រកប្រូបាបបានបាល់ទី១ខៀវ និងទី២ក្រហម។

💡 Tip: ភាគបែងថយពី ៩ ទៅ ៨។
🔑 ចម្លើយ: $P = \frac{5}{9} \times \frac{4}{8} = \frac{20}{72} = \frac{5}{18}$

លំហាត់ ២.២៖ ប្រអប់មានប៊ិចស ៦ និងខ្មៅ ២។ ទាញយក ២ បន្តបន្ទាប់មិនដាក់វិញ។ រកប្រូបាបបានប៊ិចសទាំង ២។

💡 Tip: លើកទី១បានស (ភាគយក ៦) និងលើកទី២ក៏ស (ភាគយកសល់ ៥)។
🔑 ចម្លើយ: $P = \frac{6}{8} \times \frac{5}{7} = \frac{30}{56} = \frac{15}{28}$

លំហាត់ ២.៣៖ មានកាតពណ៌បៃតង ៣ និងលឿង ៥។ ទាញ ២ បន្តបន្ទាប់មិនដាក់វិញ។ រកប្រូបាបបានបៃតង១ និងលឿង១ (មិនគិតលំដាប់)។

💡 Tip: មិនគិតលំដាប់ គឺត្រូវយក (បៃតងរួចលឿង) + (លឿងរួចបៃតង)។
🔑 ចម្លើយ: $P = (\frac{3}{8} \times \frac{5}{7}) + (\frac{5}{8} \times \frac{3}{7}) = \frac{15}{56} + \frac{15}{56} = \frac{30}{56} = \frac{15}{28}$

ក្បួនទី ៣៖ ព្រឹត្តិការណ៍ផ្ទុយ (ពាក្យ "យ៉ាងតិច")

📘 រូបមន្ត និង គោលការណ៍

នៅពេលលំហាត់សួររកប្រូបាបដែលមាន "យ៉ាងតិច១..." វិធីដែលលឿនបំផុតគឺគណនាព្រឹត្តិការណ៍ផ្ទុយរបស់វា គឺ "គ្មានសោះ"។

$P(\text{យ៉ាងតិច } 1) = 1 - P(\text{គ្មានសោះ})$

🎯 ឧទាហរណ៍គំរូ

ជ្រើសរើសមនុស្ស ៤នាក់ចេញពី បុរស ៦នាក់ និងស្ត្រី ៤នាក់។ គណនាប្រូបាបដើម្បីមានសមាជិកជាស្ត្រីយ៉ាងតិចម្នាក់។

ដំណោះស្រាយ៖

  • ព្រឹត្តិការណ៍ផ្ទុយ $\bar{A}$ គឺ "គ្មានស្ត្រីសោះ" (មានន័យថាបានបុរសទាំង ៤)។
  • ករណីអាច: $C(10,4) = 210$
  • ប្រូបាបគ្មានស្ត្រី: $P(\bar{A}) = \frac{C(6,4)}{210} = \frac{15}{210} = \frac{1}{14}$
  • ប្រូបាបយ៉ាងតិចស្ត្រី១: $P(A) = 1 - P(\bar{A}) = 1 - \frac{1}{14} = \frac{13}{14}$

✍️ លំហាត់អនុវត្តន៍ (ដូរលេខ)

លំហាត់ ៣.១៖ ថង់មួយមានបាល់ក្រហម ៥ និងខៀវ ៥។ ចាប់យក ៣។ រកប្រូបាបបានបាល់ក្រហមយ៉ាងតិច ១។

💡 Tip: ផ្ទុយគឺ "គ្មានក្រហមសោះ" គឺ "ខៀវទាំង៣"។ ករណីអាច $C(10,3)=120$។
🔑 ចម្លើយ: $1 - \frac{C(5,3)}{120} = 1 - \frac{10}{120} = \frac{110}{120} = \frac{11}{12}$

លំហាត់ ៣.២៖ ក្នុងប្រអប់មានអំពូលភ្លើងល្អ ៧ និងខូច ៣។ គេទិញ ៣ ព្រមគ្នា។ រកប្រូបាបដែលទិញប៉ះអំពូលខូចយ៉ាងតិច ១។

💡 Tip: ផ្ទុយគឺ "គ្មានខូចសោះ" គឺ "បានល្អទាំង៣"។ ករណីអាច $C(10,3)=120$។
🔑 ចម្លើយ: $1 - \frac{C(7,3)}{120} = 1 - \frac{35}{120} = 1 - \frac{7}{24} = \frac{17}{24}$

លំហាត់ ៣.៣៖ សិស្សប្រុស ៨នាក់ និងស្រី ៤នាក់។ ជ្រើសរើស ៣នាក់។ រកប្រូបាបបានសិស្សស្រីយ៉ាងតិចម្នាក់។

💡 Tip: ផ្ទុយគឺ "ប្រុសទាំង៣"។ ករណីអាច $C(12,3)=220$។
🔑 ចម្លើយ: $1 - \frac{C(8,3)}{220} = 1 - \frac{56}{220} = 1 - \frac{14}{55} = \frac{41}{55}$

ក្បួនទី ៤៖ សង្ឃឹមគណិត (Mathematical Expectation)

📘 រូបមន្ត និង គោលការណ៍

សង្ឃឹមគណិត (តម្លៃមធ្យមនៃអញ្ញាតចៃដន្យ $X$) គឺជាផលបូកនៃផលគុណតម្លៃអញ្ញាតចៃដន្យ និងប្រូបាបរបស់វា។ ប្រើញឹកញាប់ក្នុងល្បែងចាក់លុយ។

$E(X) = \sum_{i=1}^{n} x_i \cdot P(X=x_i) = x_1p_1 + x_2p_2 + ... + x_np_n$

*ប្រាក់ដែលត្រូវចាញ់ (បាត់បង់) ត្រូវសរសេរជាលេខអវិជ្ជមាន $(-)$។

🎯 ឧទាហរណ៍គំរូ

គេបោះឡុកឡាក់១គ្រាប់។ បើចេញលេខគូ អ្នកឈ្នះប្រាក់ ៣០០០រៀល តែបើចេញសេស អ្នកចាញ់ ១០០០រៀល។ គណនាសង្ឃឹមគណិត $E(X)$។

ដំណោះស្រាយ៖

  • $X$ ជាប្រាក់ទទួលបាន។ $X = \{3000, -1000\}$
  • ប្រូបាបចេញគូ: $P(\text{គូ}) = \frac{3}{6} = \frac{1}{2}$
  • ប្រូបាបចេញសេស: $P(\text{សេស}) = \frac{3}{6} = \frac{1}{2}$
  • $E(X) = (3000 \times \frac{1}{2}) + (-1000 \times \frac{1}{2}) = 1500 - 500 = 1000$ រៀល។

✍️ លំហាត់អនុវត្តន៍ (ដូរលេខ)

លំហាត់ ៤.១៖ បោះកាក់១។ បើចេញរូប (H) ឈ្នះ ៥០០០៛។ បើចេញលេខ (T) ចាញ់ ២០០០៛។ គណនាសង្ឃឹមគណិត។

💡 Tip: ប្រូបាបកាក់គឺ $1/2$ ម្នាក់។
🔑 ចម្លើយ: $E(X) = (5000 \times 0.5) + (-2000 \times 0.5) = 2500 - 1000 = 1500$ រៀល

លំហាត់ ៤.២៖ ក្នុងប្រអប់មានបាល់ក្រហម ៣ និងខ្មៅ ២។ ចាប់១បាល់។ បើបានក្រហម ឈ្នះ ៤០០០៛។ បើបានខ្មៅ ចាញ់ ១០០០៛។ គណនាសង្ឃឹមគណិត។

💡 Tip: ប្រូបាបក្រហម ៣/៥, ប្រូបាបខ្មៅ ២/៥។
🔑 ចម្លើយ: $E(X) = (4000 \times \frac{3}{5}) + (-1000 \times \frac{2}{5}) = 2400 - 400 = 2000$ រៀល

លំហាត់ ៤.៣៖ កងវិលមួយមាន ៤ផ្នែកស្មើគ្នា (ពណ៌មាស១ និងខ្មៅ៣)។ បើឈប់ចំមាស ឈ្នះ ១០០០០៛, ចំខ្មៅ ចាញ់ ២០០០៛។ គណនាសង្ឃឹមគណិត។

💡 Tip: ប្រូបាបមាស ១/៤, ប្រូបាបខ្មៅ ៣/៤។
🔑 ចម្លើយ: $E(X) = (10000 \times \frac{1}{4}) + (-2000 \times \frac{3}{4}) = 2500 - 1500 = 1000$ រៀល

ក្បួនទី ៥៖ ប្រូបាបមានលក្ខខណ្ឌ (Conditional Probability)

📘 រូបមន្ត និង គោលការណ៍

ប្រើនៅពេលគេចង់រកប្រូបាបនៃព្រឹត្តិការណ៍មួយ "ដោយដឹងជាមុនថា" ព្រឹត្តិការណ៍មួយទៀតបានកើតឡើងរួចហើយ។

$P(A|B) = \frac{P(A \cap B)}{P(B)}$ ឬ $\frac{n(A \cap B)}{n(B)}$

🎯 ឧទាហរណ៍គំរូ

សិស្ស ៥០នាក់ មាន ៣០នាក់ចេះអង់គ្លេស, ២០នាក់ចេះបារាំង និង ១០នាក់ចេះទាំងពីរ។ គេហៅសិស្សម្នាក់។ រកប្រូបាបសិស្សនោះចេះបារាំង ដោយដឹងជាមុនថាគេចេះអង់គ្លេស។

ដំណោះស្រាយ៖

  • តាង $E$: ចេះអង់គ្លេស $\Rightarrow n(E) = 30$
  • តាង $F$: ចេះបារាំង។ ចេះទាំងពីរ $\Rightarrow n(E \cap F) = 10$
  • ប្រូបាបចេះបារាំងដោយដឹងថាចេះអង់គ្លេស $P(F|E) = \frac{n(E \cap F)}{n(E)} = \frac{10}{30} = \frac{1}{3}$

✍️ លំហាត់អនុវត្តន៍ (ដូរលេខ)

លំហាត់ ៥.១៖ មនុស្ស ៤០នាក់។ ២៥នាក់ចូលចិត្តគណិត, ១៥នាក់ចូលចិត្តរូបវិទ្យា, និង ៥នាក់ចូលចិត្តទាំងពីរ។ គេជ្រើសរើសម្នាក់ដែលដឹងថាចូលចិត្តគណិត។ រកប្រូបាបដែលគាត់ចូលចិត្តរូបវិទ្យា។

💡 Tip: ភាគបែងគឺអ្នកចូលចិត្តគណិត ២៥, ភាគយកគឺអ្នកចូលចិត្តទាំងពីរ ៥។
🔑 ចម្លើយ: $P(\text{រូប}|\text{គណិត}) = \frac{5}{25} = \frac{1}{5}$

លំហាត់ ៥.២៖ ភ្ញៀវទេសចរ ៦០នាក់។ ៤០នាក់ទៅសមុទ្រ, ៣០នាក់ទៅភ្នំ, ១៥នាក់ទៅទាំងពីរ។ បើជ្រើសចំអ្នកដែលទៅសមុទ្រ តើមានប្រូបាបប៉ុន្មានដែលគាត់ទៅភ្នំដែរ?

💡 Tip: លក្ខខណ្ឌគឺ "ទៅសមុទ្រ" ៤០នាក់។
🔑 ចម្លើយ: $P(\text{ភ្នំ}|\text{សមុទ្រ}) = \frac{15}{40} = \frac{3}{8}$

លំហាត់ ៥.៣៖ ឡាន ១០០គ្រឿង។ ៦០គ្រឿងមានម៉ាស៊ីនត្រជាក់ (AC), ៥០គ្រឿងមាន GPS, ២០គ្រឿងមានទាំងពីរ។ បើគេជ្រើសឡានមួយដែលមាន AC តើប្រូបាបវាមាន GPS មានប៉ុន្មាន?

💡 Tip: លក្ខខណ្ឌគឺ "មាន AC" ៦០គ្រឿង។
🔑 ចម្លើយ: $P(GPS|AC) = \frac{20}{60} = \frac{1}{3}$
