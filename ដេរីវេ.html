មេរៀន៖ ដេរីវេនៃអនុគមន៍

គណិតវិទ្យា ថ្នាក់ទី១២

ខ្លឹមសារមេរៀនសង្ខេប

១. វិធានគ្រឹះនៃដេរីវេ

១. $(C)' = 0$ (C ជាចំនួនថេរ)
២. $(x)' = 1$
៣. $(x^n)' = n x^{n-1}$
៤. $(cx^n)' = c \cdot n x^{n-1}$
៥. $(\frac{1}{x})' = -\frac{1}{x^2}$
៦. $(\sqrt{x})' = \frac{1}{2\sqrt{x}}$
ឧទាហរណ៍៖
- $f(x) = 5 \Rightarrow f'(x) = 0$
- $f(x) = x^4 \Rightarrow f'(x) = 4x^{4-1} = 4x^3$

២. ប្រមាណវិធីលើដេរីវេ

១. ផលបូក/ដក៖ $(u \pm v)' = u' \pm v'$
២. ផលគុណ៖ $(uv)' = u'v + uv'$
៣. ផលចែក៖ $(\frac{u}{v})' = \frac{u'v - uv'}{v^2}$
៤. ចំនួនថេរគុណអនុគមន៍៖ $(cu)' = cu'$
ឧទាហរណ៍ផលគុណ៖ $y = x^2 \sin x $
$y' = (x^2)' \sin x + x^2 (\sin x)' = 2x \sin x + x^2 \cos x$

៣. អនុគមន៍ត្រីកោណមាត្រ

១. $(\sin x)' = \cos x$
២. $(\cos x)' = -\sin x$
៣. $(\tan x)' = \frac{1}{\cos^2 x} = 1 + \tan^2 x$
៤. $(\cot x)' = -\frac{1}{\sin^2 x} = -(1 + \cot^2 x)$

៤. អនុគមន៍អិចស្ប៉ូណង់ស្យែល និងលោការីត

១. $(e^x)' = e^x$
២. $(a^x)' = a^x \ln a$
៣. $(\ln x)' = \frac{1}{x}$
៤. $(\log_a x)' = \frac{1}{x \ln a}$

៥. វិធានបណ្ដាក់ (Chain Rule)

បើយើងមានអនុគមន៍បណ្ដាក់ $f(g(x))$ នោះដេរីវេគឺ៖ $[f(g(x))]' = f'(g(x)) \cdot g'(x)$

១. $(u^n)' = n \cdot u^{n-1} \cdot u'$
២. $(\sin u)' = u' \cos u$
៣. $(\cos u)' = -u' \sin u$
៤. $(e^u)' = u' e^u$
៥. $(\ln u)' = \frac{u'}{u}$
៦. $(\sqrt{u})' = \frac{u'}{2\sqrt{u}}$
ឧទាហរណ៍៖ $y = (3x+2)^4$
តាង $u = 3x+2 \Rightarrow u' = 3$
$y' = 4(3x+2)^3 \cdot (3) = 12(3x+2)^3$
